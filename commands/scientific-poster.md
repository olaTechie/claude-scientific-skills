Create a scientific poster for: $ARGUMENTS

Use the following skills (read each SKILL.md before using):

1. **Content extraction** — If a manuscript or data files are provided, extract key findings, figures, and conclusions.

2. **Poster design** — Use the pptx-posters skill for PowerPoint poster creation or the latex-posters skill for LaTeX poster creation.

3. **Figures** — Use the matplotlib, seaborn, or plotly skills to create high-resolution figures. Use the scientific-visualization skill for guidance on effective poster figures (larger fonts, simpler layouts, high contrast).

4. **Schematics** — Use the scientific-schematics skill to generate workflow diagrams, graphical abstracts, or conceptual figures.

5. **Layout** — Structure the poster with: title and authors, introduction/background, methods, key results (emphasise figures), conclusions, and references.

6. **Venue formatting** — Use the venue-templates skill if a specific conference template is needed.

Save the poster file to the working directory.
