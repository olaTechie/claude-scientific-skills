Conduct a comprehensive literature review on: $ARGUMENTS

Use the following skills (read each SKILL.md before using):

1. **Search strategy** — Use the literature-review skill for methodology guidance. Define inclusion/exclusion criteria and search terms.

2. **Database searches** — Use the pubmed-database skill to search PubMed for peer-reviewed articles. Use the biorxiv-database skill for recent preprints. Use the openalex-database skill for broader academic literature coverage.

3. **Paper screening** — Screen titles and abstracts against the inclusion criteria. Categorise papers by relevance (high/medium/low).

4. **Full-text analysis** — For highly relevant papers, extract key findings, methods, sample sizes, and conclusions.

5. **Evidence synthesis** — Identify themes, consensus findings, contradictions, and gaps in the literature. Note methodological strengths and limitations across studies.

6. **Citation management** — Use the citation-management skill to organise references and generate a bibliography.

7. **Report generation** — Use the scientific-writing skill to draft a structured literature review with: introduction and scope, search methodology, thematic synthesis of findings, identified gaps and future directions, and a complete reference list.

Save the review as a markdown file. If requested, use the docx document skill to create a Word document.
