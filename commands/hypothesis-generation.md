Generate and evaluate research hypotheses for: $ARGUMENTS

Use the following skills (read each SKILL.md before using):

1. **Brainstorming** — Use the scientific-brainstorming skill for structured ideation. Use the hypothesis-generation skill (and the hypogenic skill if appropriate) for systematic hypothesis formulation.

2. **Literature grounding** — Use the pubmed-database and openalex-database skills to check what is already known. Identify gaps in current understanding.

3. **Data support** — If data is available, use the exploratory-data-analysis and statistical-analysis skills to identify patterns that support or challenge hypotheses.

4. **Database evidence** — Query relevant databases to gather supporting evidence: gene-database, uniprot-database, kegg-database, string-database, or others as appropriate for the domain.

5. **Critical evaluation** — Use the scientific-critical-thinking skill to evaluate each hypothesis for: testability, falsifiability, novelty, feasibility, and potential impact.

6. **Experimental design** — For the most promising hypotheses, outline experimental approaches to test them, including required methods, sample sizes, and expected outcomes.

7. **Output** — Generate a ranked list of hypotheses with: statement, supporting evidence, testability assessment, proposed experiments, and confidence level.

Save the hypothesis report to the working directory.
