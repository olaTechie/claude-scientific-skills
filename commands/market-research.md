Conduct market research and competitive intelligence for: $ARGUMENTS

Use the following skills (read each SKILL.md before using):

1. **Clinical landscape** — Use the clinicaltrials-database skill to map the clinical trial landscape: trial counts by phase, active sponsors, endpoints being used.

2. **Drug pipeline** — Use the chembl-database skill to identify compounds in development. Use the drugbank-database skill for approved drug information. Use the fda-database skill for regulatory actions.

3. **Patent landscape** — Use the uspto-database skill to search for relevant patents, identify key patent holders, and assess freedom to operate.

4. **Scientific literature** — Use the pubmed-database and openalex-database skills to track publication trends, key opinion leaders, and emerging research themes.

5. **Financial data** (if applicable) — Use the edgartools skill for SEC filings of public pharma/biotech companies. Use the alpha-vantage skill for stock data and financial metrics.

6. **Report** — Use the market-research-reports skill for structure and formatting guidance. Generate a comprehensive market research report with: market overview, competitive landscape, pipeline analysis, patent summary, key trends, and strategic implications.

Save the report to the working directory.
