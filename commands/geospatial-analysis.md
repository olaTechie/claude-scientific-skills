Perform geospatial analysis for: $ARGUMENTS

Use the following skills (read each SKILL.md before using):

1. **Data loading** — Use the geopandas skill to load and manipulate geospatial data (shapefiles, GeoJSON, GeoPackage, CSV with coordinates).

2. **Public data** — Use the datacommons-client skill to retrieve public datasets from Data Commons (demographics, health, economics) for spatial context.

3. **Spatial analysis** — Use the geopandas skill for: spatial joins, buffering, overlay operations, distance calculations, and spatial statistics.

4. **Visualisation** — Use the matplotlib and plotly skills for choropleth maps, point maps, and spatial distribution plots. Use the geopandas built-in plotting for quick maps.

5. **Statistical analysis** — Use the statsmodels skill for spatial regression or the scikit-learn skill for spatial clustering and classification.

6. **Report** — Generate maps and a summary of spatial patterns, clusters, and key geographic findings.

Save all maps, data, and analysis to the working directory.
