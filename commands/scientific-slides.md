Create a scientific presentation for: $ARGUMENTS

Use the following skills (read each SKILL.md before using):

1. **Content planning** — Structure the talk with a clear narrative arc: motivation, background, methods, results, and conclusions.

2. **Slide creation** — Use the scientific-slides skill for guidance on effective scientific presentations. Use the pptx document skill to create the PowerPoint file.

3. **Figures & visualisations** — Use the matplotlib, seaborn, or plotly skills for data figures. Use the scientific-schematics skill for conceptual diagrams and workflow figures.

4. **Speaker notes** — Add detailed speaker notes for each slide with key talking points and timing guidance.

5. **References** — Use the citation-management skill to add a references slide with key citations.

Save the presentation file to the working directory.
